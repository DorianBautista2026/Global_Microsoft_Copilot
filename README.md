# Publicación en GitHub Pages

Este paquete está listo para subirse a la raíz del repositorio de GitHub Pages.

Contenido:

- `index.html`: dashboard principal.
- `global_maz_cross/Cruce_Global_MAZ_Copilot_Marzo_vs_May2026.xlsx`: Excel descargable desde el botón del dashboard.

Pasos:

1. Abre el repositorio en GitHub.
2. Entra a la rama `main`.
3. Sube `index.html` reemplazando el actual.
4. Sube la carpeta `global_maz_cross` completa.
5. Haz commit con un mensaje como `Actualizar dashboard cruce Global MAZ Copilot`.
6. Espera 1-3 minutos a que GitHub Pages publique el cambio.
